Illustrative synthetic data — not actual results

This package contains fictional synthetic survey data for a dashboard mock-up. It is not actual results and should not be interpreted as Vancity performance.

Recommended Power BI model:
- synthetic_survey_responses.csv: one row per synthetic survey response.
- synthetic_behaviour_statement_responses.csv: one row per response per behaviour statement for matrix/heatmap visuals.
- value_add_response_outcomes_long.csv: one row per selected value-add outcome for multi-select analysis.
- Summary CSVs: pre-aggregated tables for a PowerPoint mock-up.
